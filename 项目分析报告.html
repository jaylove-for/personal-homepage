4. 关键知识点讲解

4.1 配置驱动 (Configuration-Driven)

传统建站：改页面内容 → 改代码 → 重新部署。
PRISM 方式：改 TOML 文件 → 重新部署。代码原封不动。

举例：你想在导航栏加一个"项目"页面，只需要：
① 新建 content/projects.toml 文件
② 在 config.toml 的 [[navigation]] 里加一条
③ 完成。不需要碰任何 .tsx 代码。

4.2 静态站点生成 (SSG)

PRISM 使用 Next.js 的静态导出（Static Export）功能。所有页面在 npm run build 时预渲染为纯 HTML/CSS/JS 文件，输出到 out/ 目录。

好处：加载极快（无服务端渲染延迟）、免费部署（GitHub Pages）、无数据库维护成本。

4.3 国际化 (i18n) 机制

PRISM 通过两个目录实现中英文切换：

  • content/ — 默认语言（英文）
  • content_zh/ — 中文内容

读取时，系统先查找对应语言的文件，找不到就回退到默认。所以你不需要两份完整的文件，只翻译不同的部分即可。

4.4 BibTeX → 论文列表

从 Google Scholar / Zotero 导出 .bib 文件，放到 content/ 目录。PRISM 自动解析并生成带搜索、过滤的论文列表页面。支持按年份、类型筛选。

4.5 动态路由 [slug]

src/app/[slug]/page.tsx 是一个通配路由。访问 /cv 时，Next.js 读取 content/cv.toml，根据配置的 type（text/card/publication）渲染对应组件。

5. 个性化改编计划

以下是将 PRISM 改造为钟杰个人学术主页的具体步骤：

1
修改全局配置

编辑 content/config.toml：名字 → 钟杰，Title → 医学信息工程本科生 / 副教授助理，机构 → 成都中医药大学，社交链接 → GitHub / 抖音

2
撰写个人简介

编辑 content/bio.md：介绍自己的学术背景、研究方向（中医药信息化）、科研经历（副教授助理、论文参与）

3
添加论文数据

你的三作和四作论文信息写入 content/publications.bib

4
定制导航菜单

精简导航：关于我 → 论文 → 项目 → 简历，去掉 Teaching/Services 等不适用模块

5
中文优先

设置 default_locale = "zh"，让中文成为默认语言

6
视觉微调

调整 Tailwind 主题色，加入中医药元素配色（如墨绿/暖棕基调）

7
部署上线

GitHub Pages 免费部署，绑定自定义域名（可选）
